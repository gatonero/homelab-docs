# Homelab Documentation

This repository contains structured documentation of a personal homelab.

The focus is on architecture, networking, services, and operations, with an emphasis
on clarity, reproducibility, and long-term maintainability. The documentation applies
production-style thinking to a homelab environment.

## Scope

The documentation covers:

- overall architecture and design principles
- networking, DNS, TLS, and access models
- virtualization and container platforms
- service documentation and onboarding
- backup, restore, and recovery procedures
- operational policies and drills
- NetBox as the central source of truth

## Intended Audience

This documentation is written for technically experienced administrators.
It is not a step-by-step beginner tutorial.

It is intended to serve as:
- a technical reference
- a decision log
- a long-term knowledge base
