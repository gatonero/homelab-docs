# E-Ink Roadmap

This document describes the planned evolution of the Homelab E-Ink platform.

---

## Completed

- Home Assistant dashboard
- Puppet evaluation
- ESPHome evaluation
- Docker proxy evaluation
- Cleanup of evaluation artifacts

---

## In Progress

- Documentation
- Version control

---

## Planned

### Phase 1

Integrate the Seeed Studio XIAO 7.5 into the TRMNL Cloud.

Objectives:

- Native TRMNL workflow
- Shared dashboards
- Stable firmware
- Reliable update cycle

---

### Phase 2

Deploy TRMNL BYOS.

Objectives:

- Fully self-hosted deployment
- Docker-based installation
- Local device management

---

### Phase 3

Migrate all E-Ink devices to BYOS.

Objectives:

- Remove cloud dependency
- Unified rendering pipeline
- Centralized device management

---

## Long-Term Vision

The Homelab will provide a fully self-hosted E-Ink platform in which Home Assistant acts as the data source, TRMNL BYOS performs rendering and device management, and multiple E-Ink devices present the same information through a common rendering pipeline.
