# Homelab Markup Library

## Ziel

Diese Bibliothek enthält wiederverwendbare Blade-Bausteine für Larapaper BYOS.

Sie basiert ausschließlich auf verifizierten TRMNL-Komponenten.

---

# Architektur

```
Home Assistant
        │
        ▼
JSON Payload
        │
        ▼
Recipe
        │
        ▼
Homelab Components
        │
        ▼
TRMNL Components
        │
        ▼
plugins.css
        │
        ▼
PNG
```
