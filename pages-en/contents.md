# Contents

This document provides an index of the homelab documentation.

It serves as a navigational overview of all documentation sections under `pages/`.

---

## Overview

The documentation is organized into the following main areas:

* **Overview** – high-level orientation and guiding principles
* **Architecture** – structural and architectural documentation
* **Networking / DNS / TLS** – access, naming, and security model
* **Services** – service descriptions and operational context
* **Operations** – procedures, policies, and drills
* **NetBox** – source of truth and working model
* **Archive** – historical and deprecated documents

---

## Navigation

Each section contains:

* an overview document
* detailed sub-documents
* cross-references where applicable

The documentation is intended to be read top-down or selectively, depending on context.

---

## Conventions

* Naming conventions apply across all sections
* Terminology is used consistently
* NetBox is the authoritative source for structural data

---

## Usage

This documentation is intended as:

* a reference for operations
* a decision log
* a long-term knowledge base

It prioritizes clarity, consistency, and reproducibility.
